# Cartesian Product Problem

## Question

Generate combinations from N number of arrays with M number of elements in them, in JavaScript & PHP.

```html
X = {a, b, c}
Y = {p, q}
Z = {r, s}

Define a Operation,

X * Y * Z = {
    {a, p, r}, 
    {a, p, s}, 
    {a, q, r}, 
    {a, q, s},
    ...
    {c, q, s}
}
```
